# Smart Academic Task Organizer

An AI-powered web application that helps university students automatically extract timetable and assignment information, summarize assignment requirements, and generate optimized study schedules.

## Overview

Smart Academic Task Organizer reduces the manual effort of academic planning by integrating OCR, Natural Language Processing (NLP), Large Language Models (LLMs), and scheduling optimization into a single web application.

Users can upload timetable images and assignment PDF files, and the system automatically:

- Extracts timetable information from images
- Extracts assignment metadata from PDF documents
- Generates concise assignment summaries
- Creates an optimized study schedule based on deadlines and timetable constraints

---

## Features

### Timetable Extraction
- Upload timetable images
- OCR-based text recognition
- Automatic timetable parsing
- Weekly timetable visualization

### Assignment Processing
- Upload assignment PDF files
- Automatic metadata extraction
- AI-generated assignment summaries
- Task breakdown

### Intelligent Scheduler
- Deadline-aware scheduling
- Workload balancing
- Conflict avoidance
- Weekly calendar generation
